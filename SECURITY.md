# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public issue.

Instead, send an email to security@investment-agent.ai with:
- Description of the vulnerability
- Steps to reproduce (if applicable)
- Potential impact assessment
- Suggested fix (if any)

We will respond within 48 hours and work with you to assess and address the issue.

## Security Considerations

- This framework is for **research and educational purposes**.
- Never commit API keys, passwords, or trading credentials to the repository.
- Use environment variables or secure vaults for sensitive configuration.
- Review third-party data provider terms of service before use.
